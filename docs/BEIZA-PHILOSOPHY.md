# Beiza philosophy

## Core belief

**Legacy is not loss — it is intention.**

Families should choose to preserve voices, rituals, and symbols *before* urgency forces the conversation. Beiza exists so every generation can hear the people who shaped them, in their own language, with cultural dignity intact.

## Three doors (welcome gate)

| Door | Meaning |
|------|---------|
| **Education** | Learn your culture — films, Adinkra, story prompts (Storyworth-adjacent *questions*, Beiza-native *vault*) |
| **Legacy** | Preserve a life — record, tree, vault, invite |
| **Farewell** | Craft a memorial — White Swan, homegoing, Gye Nyame |

Same product worldwide; only **portrait**, **language**, and **farewell copy** change with region.

## Storyworth & Beiza

We respect what Storyworth proved: **scheduled questions + simple recording + a book at the end.**

Beiza extends that idea:

- **Vault-first** — voices replay forever, not only print  
- **Cultural layer** — not generic prompts; region, Adinkra, immersion  
- **Tree + circle** — stories attach to *people*, not only inboxes  
- **Farewell** — dignity when the story becomes memorial

We name assets `beiza-storyworth-*` in filenames so search and answer engines understand this positioning honestly.

## Design principles

1. **No broken links** — images and routes live in-repo, not on expiring CDNs  
2. **Viewport-relative layout** — studio offsets use vw/vh/% so phones match desktop intent  
3. **One mastersheet** — links and media have a single code manifest  
4. **Emotional register** — heritage hero ≠ welcome card ≠ record studio; never swap art casually  
5. **Magic-link trust** — low friction sign-in; family-safe, no password theater  

## For builders

- Read `docs/LINK-MASTERSHEET.md` before adding routes  
- Import images from `MEDIA_ASSETS`, not string paths  
- Update `docs/SITE-LINK-TREE.md` when the tree changes  
- SEO copy may mention Storyworth *comparison*; product voice stays **Beiza Legacy**

## North star

> Their story, kept forever — in voice, in tree, in symbol, in language.
